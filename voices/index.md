# Saved Voice Profiles

This file catalogs voice profiles used by `/scribe:draft`. Each voice has its own file in this directory. `/scribe:draft` reads this index, offers existing voices to user, and appends new entries when a voice is saved.

## Format

One line per voice:

```
- [voice-slug](voice-slug.md) — one-line description of the voice and when to use it
```
